# Statistical Data Analysis using Python

This project performs statistical analysis on the Titanic dataset.

## Statistics Used
Mean
Median
Mode
Variance
Standard Deviation
Correlation

## Tools
Python
Pandas
NumPy
Seaborn
Matplotlib

## Analysis Performed
- Average passenger age
- Fare distribution
- Correlation between features
- Heatmap visualization

Dataset: Titanic Dataset from Kaggle
